Getting Started

1) Installing the Printer Driver

Notes:
All applications must be closed before installing or upgrading the driver.
This procedure requires administrator privileges (admin account or "Run as Administrator").
As the information contained in this document applies to multiple print devices, the word "device" is used as a generic reference to whichever device is being used.

During installation, a dialog window will appear that will prompt for the generic name (PCL6 V4 Driver for Universal Print) or a version-specific name (RICOH PCL6 V4 UniversalDriver VX.X) of the driver. The version-specific name of the driver cannot be upgraded to a new version of the driver because the naming between the driver with the new version-specific name and the old one is different. Therefore, if there are plans to upgrade the driver to a later version in the future, please select the driver with the generic name.

(1)-1 For devices connected via USB

(1)-1-1 Windows 8.1/10/Server 2012/Server 2012 R2/Server 2016/Server 2019

  1. Connect the "Device" to a PC using a USB cable and then power on the "Device".
  2. Automatic installation of the device driver will appear to start and then fail, displaying the message "Device driver software was not successfully installed". Close this message.
  3. Open "Device Manager".
  4. The "Device" should be listed under "Other devices". Right-click "Device" and select "Update Device Software...".
  5. When the "Update Device Software" screen is displayed, select "Browse my computer for driver software".
  6. Browse to the folder containing the driver files (typically the CD-ROM or an extracted archive downloaded from the website) and click "OK".
  7. Follow the rest of the Wizard's instructions to complete the installation.

(1)-2 For devices connected via network

(1)-2-1 Windows 10/Server 2016/Server 2019

  1. From the Start menu, select "Settings" > "Devices" > "Printers & scanners" > "Add printers & scanners" or select "All Apps" > "Windows System" > "Control Panel" > "View devices and printers" > "Add a printer".
  2. When the window for adding a printer appears, displaying the message "Searching for printers & scanners...", click "The printer that I want isn't listed".
  3. In the "Find a printer by other options" window, select "Add a local printer or network printer with manual settings" and then click "Next".
  4. In the "Choose a printer port" window, select "Create a new port:", choose "Standard TCP/IP Port" as the "Type of port:", and then click "Next".
  5. In the "Type a printer hostname or IP address" window, enter the hostname or IP address in "Hostname or IP address", uncheck "Query the printer and automatically select the driver to use", and then click "Next".
  6. When the "Install the printer driver" window appears, click "Have Disk".
  7. Browse to the folder containing the driver files, which are typically on the device's installation from the website as an executable file, and then click "OK".
  8. Follow the wizard's instructions to complete the installation.

(1)-2-2 Windows 8.1/Server 2012/Server 2012 R2

  1. From the Charms Bar, select Settings > Control Panel > View devices and printers > Add a printer.
  2. When the "Add a Printer" window appears, displaying the message "Searching for available printer...", click "The printer that I want isn't listed".
  3. In the "Find a printer by other options" window, select "Add a local printer or network printer with manual settings" and then click "Next".
  4. In the "Choose a printer port" window, select "Create a new port:", choose "Standard TCP/IP Port" as the "Type of port:", and then click "Next".
  5. In the "Type a printer hostname or IP address" window, enter the hostname or IP address in "Hostname or IP address", uncheck "Query the printer and automatically select the driver to use", and then click "Next".
  6. When the "Install the printer driver" window appears, click "Have Disk".
  7. Browse to the folder containing the driver files, which are typically on the device's installation from the website as an executable file, and then click "OK".
  8. Follow the wizard's instructions to complete the installation.

2) Upgrading the printer driver

Please conduct the following procedure to upgrade the printer driver. This procedure requires administrator privileges. Close all applications before starting the upgrade process.
Attention: When upgrading the printer driver, the same printer driver name must be used.
  1. In the "Control Panel", navigate to "View devices and printers" and open "Devices and Printers".
  2. Open "Printer Properties" of the printer to be upgraded.
  3. Select the "New Driver" button on the "Advanced" tab. In the "Add Printer Driver Wizard", click "Next".
  4. Click "Have Disk". Browse to the INF file in the driver package, select it, and click "OK".
  5. Select the appropriate manufacturer and printer and click "Next".
  6. When the installation is complete, click "Finish".
  7. Click "OK" to close "Printer Properties".
  8. Log off and log on again. After completing this operation, the new version of the printer driver can be used.

3) Uninstalling the printer driver

Note: This procedure requires administrator privileges (admin account or "Run as Administrator").

3-1) Windows 8.1/10/Server 2012/Server 2012 R2/Server 2016/Server 2019

  1. Close all applications.
  2. Open the "Printers" folder (or "Devices and Printers").
  3. Delete the printer icon corresponding to the driver that is to be uninstalled.
  4. Print server properties > "Driver" tab.
  5. Press the Change Driver Settings button.
  6. Select the driver to be uninstalled.
  7. Press the Remove key.
  8. Press "Remove driver and driver package".
  9. Restart the PC.

    • Staple
    • Punch
    • Fold
    • Sort
    • Perfect Binding
    • Layout and Booklet etc.
      The example of specific application : Microsoft Office
  • In the environment to which the computer name is set by 16 bytes or more, the host name with banner page print is printed only up to 15 bytes.
  • On some applications, the backside of a duplex page is incremented by the device counter. These applications add a blank page to the end of odd-paged duplex jobs.

      Workaround:
      Specify the number of copies using the driver.
  • Use of Raster Mode with the Layout settings (= pages per sheet) may be printed roughly.

      Workaround:
      Use Vector Mode in the PCL driver.
  • If the registry and configuration files that are generated during driver installation are edited or deleted, the behavior of the driver is not guaranteed.
  • It might take a significant amount time to open "More settings" in the Print Dialog in an application from the Windows Store.
  • When enabling Poster, blank paper might be printed even if Do not Print Blank Pages is enabled.
  • When selecting the file path in Windows Store applications, it is necessary to enter the path manually.
  • The text settings of the Ricoh Driver Utility might behave in the following ways:
    1. The text might be empty when entering double-byte characters.
    2. The cursor position will return to the beginning of the input characters when entering text as a password by copying and pasting or copying by holding the SHIFT key.
    3. The text will not change if copying and pasting a number of characters that exceed the maximum character limit.
  • The preview of the watermark will display even if the driver can't print the fonts as a watermark.
  • Print jobs may be canceled when specific data with including a lot of graphics data is printed.
  • When two or more printer drivers share the same USB port, USB bidirectional communication may fail.
  • Only the user who installed the printer driver can use the following functions.
    • Use "Printer Properties".
    • Edit Shared "One Click Preset Name/Comment" on "Manage One Click Prest List".
    • Register "One Click Preset" with "Do not Allow User to Register Preset",
  • If you specify the Web browser of the store application such as Microsoft Edge as the default application, if you try to refer to the help of the printer driver from that browser, the help page is displayed to yourself and you can not refer to the contents.
    In that case, change the default Web browser or see the help of the printer driver from the desktop application.
  • If the printer times out during printing and printing can not be performed, set the network timeout setting of the printer longer (300 seconds, etc.).
  • When printing using [layout] and [page frame border] function, the underline of the border is printed thin.
  • When printing using [Choose paper source by PDF page size] setting from the following applications, a different paper size from print setting might be used:
    • Adobe Reader
    • Adobe Acrobat Reader DC
    In that case, change [Choose paper source by PDF page size] setting.
  • The following issues occurred before the firmware update:
    • When printing in Raster Mode, part of the document might be missing.
    • When printing with the Watermark function, the output of the watermark might be different from the preview.
    • When printing with the Front Cover function, only the front cover might be printed and not the rest of the document.
    • When printing with the Toner Save function, some images might be printed without toner saving.
    • When printing with both the Watermark and Toner Save functions, some characters might be printed in bold.
    • Network bidirectional communication might not work correctly for the DL Env. setting and the Custom Paper Size setting.
    Notes: Target devices and the fixed firmware versions:
    • SP 330DN/SP 3710DN/330DN/3710DN: v1.04 or later
    • SP 330SN/SP 330SFN/SP 3710SF/330SN/330SFN/3710SF: v1.04 or later

Because of this, the functions that require user information such as "User ID", "User Code", and "Password" will not work.

  • If Printing Preferences is launched soon after restarting the computer, the Windows standard UI might be displayed instead of the Ricoh extended UI. If this occurs, wait for a while and then relaunch Printing Preferences.
  • In Windows 10 or Server 2016, if the "Esc" key is pressed on the Store Device App launched from "More settings" in the Print Dialog in an application from the Windows Store, the setting values are saved and the screen is closed in the same manner as if the "OK" button were pressed.

  • Some shaded images may not be printed correctly when Raster Mode is used.

      Workaround:
